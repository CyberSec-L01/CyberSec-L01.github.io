4. KMP

目的利用字符串的前后缀匹配以使得不用每次回退到从第一位开始比较，next 数组为匹配字符串的前后缀匹配位数，判断要回退几位，其求法与实际匹配字符串一致。为了减少边界判断，对字符串加 ' ' 处理，即字符串从1开始。

5. trie树

若想用数组实现，需要维护一个数组 son[N][26] 存储每个节点的子节点, cnt[N] 以某节点结尾的单词数量,idx索引;
